Media Sources

This app fetches open-licensed images and sounds from Openverse.

Back to app

  • Images are fetched live from the Openverse image API and usually come from sources like Flickr or Wikimedia Commons.
  • Sounds are fetched live from the Openverse audio API and often come from sources like Freesound or Wikimedia Commons.
  • Each result keeps its original creator, license, and source page metadata inside the Openverse result.
  • Openverse: https://openverse.org/
  • Openverse API docs: https://api.openverse.org/v1/
