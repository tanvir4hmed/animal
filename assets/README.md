# Asset Slots

Drop your real images, GIFs, and original sound clips here.

## Folders

- `categories/` for category cover pictures
- `gifs/` for animal GIFs or moving WebP files
- `sounds/` for real animal sound clips

## Example names

- `categories/farm.jpg`
- `categories/wild.jpg`
- `categories/birds.jpg`
- `gifs/cow.gif`
- `gifs/tiger.gif`
- `gifs/parrot.webp`
- `sounds/cow.mp3`
- `sounds/tiger.wav`
- `sounds/bee.ogg`
