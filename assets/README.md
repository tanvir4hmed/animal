# Asset Slots

Drop your own better media here when you are ready.

## Folders

- `images/` for real still photos
- `gifs/` for motion visuals
- `sounds/` for real audio clips

## Example names

- `images/cow.jpg`
- `images/tiger.webp`
- `gifs/bee.gif`
- `sounds/cow.mp3`
- `sounds/rain.wav`
